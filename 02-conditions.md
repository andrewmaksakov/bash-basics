# 02 — Условия (if / else)

## Что это такое

**Условие** — проверка, по результату которой выполняется один или другой блок кода.

**`if`** — ключевое слово которое начинает условие. По-русски: "если".

**`else`** — блок, который выполнится если условие НЕ выполнилось. По-русски: "иначе".

**`elif`** — "иначе если", дополнительная проверка если предыдущее условие не сработало.

## Для чего нужно

- Чтобы скрипт **принимал решения** в зависимости от ситуации.
- Без условий: скрипт всегда делает одно и то же.
- С условиями: "если файл существует — обработай его, иначе — выведи ошибку".

**Примеры в реальной жизни:**
- Проверить что аргумент передан, иначе выйти с ошибкой
- Проверить что файл существует, прежде чем его читать
- Проверить exit code предыдущей команды и решить что делать дальше

---

## Базовый синтаксис — как пользоваться

```bash
if [[ условие ]]; then
    команды
fi
```

**С else:**
```bash
if [[ $1 == "start" ]]; then
    echo "запускаем"
else
    echo "неизвестная команда"
fi
```

**С elif:**
```bash
if [[ $1 == "start" ]]; then
    echo "запускаем"
elif [[ $1 == "stop" ]]; then
    echo "останавливаем"
else
    echo "неизвестная команда"
fi
```

**Важно:**
- Пробелы внутри `[[ ]]` обязательны: `[[ $a == "x" ]]`, не `[[$a=="x"]]`
- В конце обязательно `fi` (это `if` наоборот — закрывает блок)
- После условия точка с запятой и `then`

## Операторы сравнения

### Строки

| Оператор | Что значит |
|----------|-----------|
| `==` | Равно |
| `!=` | Не равно |
| `-z` | Строка пустая |
| `-n` | Строка не пустая |

```bash
if [[ $NAME == "Andrew" ]]; then ...
if [[ $NAME != "Bob" ]]; then ...
if [[ -z $NAME ]]; then echo "имя не задано"; fi
```

### Числа

| Оператор | Что значит | По-английски |
|----------|-----------|--------------|
| `-eq` | Равно | equal |
| `-ne` | Не равно | not equal |
| `-lt` | Меньше | less than |
| `-le` | Меньше или равно | less or equal |
| `-gt` | Больше | greater than |
| `-ge` | Больше или равно | greater or equal |

```bash
if [[ $AGE -gt 18 ]]; then echo "взрослый"; fi
if [[ $COUNT -eq 0 ]]; then echo "пусто"; fi
```

### Файлы

| Оператор | Что значит |
|----------|-----------|
| `-e file` | Файл существует (любого типа) |
| `-f file` | Файл существует и это обычный файл |
| `-d file` | Существует и это директория |
| `-r file` | Файл существует и доступен на чтение |
| `-w file` | Файл существует и доступен на запись |
| `-x file` | Файл существует и исполняемый |

```bash
if [[ -f /etc/passwd ]]; then echo "файл есть"; fi
if [[ ! -f /tmp/lock ]]; then echo "лока нет"; fi
```

`!` перед условием — отрицание ("НЕ").

---

## Логические операторы

| Оператор | Что значит |
|----------|-----------|
| `&&` | И (оба условия должны быть истинны) |
| `\|\|` | ИЛИ (хотя бы одно истинно) |
| `!` | НЕ (инвертирует) |

**Внутри `[[ ]]`:**
```bash
if [[ $AGE -gt 18 && $AGE -lt 65 ]]; then echo "работоспособный"; fi
if [[ $NAME == "Andrew" || $NAME == "Bob" ]]; then echo "знакомый"; fi
```

**Между командами:**
```bash
mkdir /tmp/test && echo "создано"     # echo выполнится только если mkdir успешен
mkdir /tmp/test || echo "ошибка"      # echo выполнится только если mkdir упал
```

## Шпаргалка

```bash
# Базовая конструкция
if [[ условие ]]; then
    команды
elif [[ условие2 ]]; then
    команды
else
    команды
fi

# Строки
[[ $a == "x" ]]       # равно
[[ $a != "x" ]]       # не равно
[[ -z $a ]]           # пустая
[[ -n $a ]]           # не пустая

# Числа
[[ $n -eq 5 ]]        # ==
[[ $n -ne 5 ]]        # !=
[[ $n -lt 5 ]]        # <
[[ $n -le 5 ]]        # <=
[[ $n -gt 5 ]]        # >
[[ $n -ge 5 ]]        # >=

# Файлы
[[ -e file ]]         # существует
[[ -f file ]]         # обычный файл
[[ -d dir ]]          # директория
[[ ! -f file ]]       # НЕ существует

# Логика
[[ a && b ]]          # И
[[ a || b ]]          # ИЛИ
[[ ! a ]]             # НЕ

# Сокращённо между командами
команда1 && команда2  # если 1 успех — делать 2
команда1 || команда2  # если 1 упала — делать 2
```

## Полный пример

```bash
#!/bin/bash
# Запуск: bash script.sh /etc/passwd

FILE=$1

# Проверить что аргумент передан
if [[ -z $FILE ]]; then
    echo "Использование: bash script.sh <файл>"
    exit 1
fi

# Проверить что файл существует и доступен на чтение
if [[ ! -f $FILE ]]; then
    echo "Ошибка: файл $FILE не найден"
    exit 1
fi

if [[ ! -r $FILE ]]; then
    echo "Ошибка: нет прав на чтение $FILE"
    exit 1
fi

# Посчитать строки
LINES=$(wc -l < "$FILE")

if [[ $LINES -gt 100 ]]; then
    echo "Большой файл: $LINES строк"
elif [[ $LINES -gt 0 ]]; then
    echo "Маленький файл: $LINES строк"
else
    echo "Файл пустой"
fi
```
