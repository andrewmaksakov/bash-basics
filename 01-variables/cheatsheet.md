# Шпаргалка — Переменные

## Переменные

```bash
NAME="Andrew"      # создать
echo "$NAME"       # прочитать
```

## Аргументы скрипта

```bash
$0    # имя скрипта
$1    # первый аргумент
$2    # второй
$#    # количество аргументов
$@    # все аргументы
```

## Exit code

```bash
$?         # код выхода последней команды
exit 0     # успех
exit 1     # ошибка
```

## Редирект в файл

```bash
echo "x" > file       # перезаписать
echo "x" >> file      # добавить в конец
echo "x" 2> err.log   # только ошибки
```

## Команда внутри строки

```bash
$(date)     # текущая дата
$(whoami)   # текущий юзер
$(pwd)      # текущая папка
```

## Определения

| Термин | Определение |
|--------|------------|
| Переменная | Именованная область памяти, в которой хранится значение |
| Аргумент | Значение, которое передаётся скрипту при запуске |
| Exit code | Число, которое возвращает программа после выполнения (0 = успех, не 0 = ошибка) |
